# 🛒 Dashboard de Vendas de E-Commerce

Este repositório contém a documentação, os links de acesso e a apresentação de um Dashboard interativo de Vendas de E-Commerce desenvolvido no **Looker Studio**, utilizando dados obtidos originalmente no Kaggle e tratados no Google Sheets.

---

## 📊 Visão Geral do Dashboard
O dashboard foi estruturado para fornecer uma visão gerencial e analítica sobre o desempenho de vendas, lucros e comportamento de pedidos de um e-commerce.

🔗 **[Acessar Dashboard Interativo (Looker Studio)](https://datastudio.google.com/u/0/reporting/a2096fa5-9934-4a33-928d-de2f7f095dae/page/gME4F/edit)**  
📁 **[Acessar Planilha de Dados Tratados (Google Sheets)](https://docs.google.com/spreadsheets/d/1nEhYmkrT3U3_KiVCNqoQgabC0hvhdFikYRcrN4raoxA/edit?usp=sharing)**

---

## 📈 Principais Métricas e Indicadores (KPIs)
O painel monitora métricas essenciais para a tomada de decisão:
* **Total de Vendas / Montante Total**
* **Total de Lucro**
* **Total de Pedidos**
* **Análise por Categoria de Produtos** (Electronics, Home, Fashion, Sports, etc.)
* **Desempenho por Região** (South, North, East, West, Central)
* **Linha Temporal de Vendas** (evolução diária/período)
* * **Vendas por Gênero:** Análise de comportamento de compra segmentada por público (Feminino, Masculino e Outros), permitindo entender a distribuição da base de clientes do e-commerce.

---

## 🛠️ Ferramentas e Tecnologias Utilizadas
* **Google Planilhas (Google Sheets):** Armazenamento e estruturação da base de dados tratada.
* **Looker Studio:** Construção dos gráficos, filtros interativos e design do painel.
* **Kaggle:** Fonte dos dados brutos.
